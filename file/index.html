1강. 변수

  • var : 변수를 선언하는 키워드

    ex)var name="변수명(문자열)"
    var는 ES6이 나온 이후로 비권장

  • let : 변수의 값이 빈번하게 변경이 일어날떄 사용

    let score=0;
    score-score+1;
    console.log(score)

  • const : 변수의 값이 변화가 없을때 사용
    • 값이 변화하지 않기 때문에 예측가능하다
    • 블럭단위이기 때문에 블럭안에서 작성된 변수는 블럭 밖에서 호출할 수 없다.
    • 블럭밖에서 선언 되더라고 블럭 위에서 선언되면 아래로 타고 내려올 수 있다. (뭔소리야)

2강. 데이터타입

  • String(문자) ex const name = "scalper" 문자열은 쌍따옴표("") 안에 선언한다
  • Number(숫자) ex const age="30"
  • boolean(참거짓) ex const is Mail = true; const isFemale =flase;
  • null(값을 비워놓은것) ex let money = null;
  • undefined(값이없는거) ex let girlFriend = undefinded;

sample Code 1

const name = "scalper";
const age="30";
const isMale = true;
const isFemale = false;
let money = null;
let girlFriend = undefined;

sample Code 1 출력예

console.log(typeof name)
console.log(typeof money)
console.log(typeof girlFriend)
console.log(name, age, isMale, money)
문장으로 넣고 싶을때-옛날방식
console.log("my name is " + name + " and i am " + age)
문장으로 넣고 싶을때-ES6 템플릿
console.log(`내 이름은 ${name} 이고 나이는 ${age}입니다.`)

sample Code 2 > substring

문자열을 중간에 자르는거(문자열의 인덱스는 0부터 시작)
const animal="tiger"
console.log(animal.substring(0,2))
console.log(animal.substring(2,5)) 두번째 인덱스부터 5번 앞 인덱스까지 출력해라는 뜻

sample Code 3 > split

문자열을 쪼개는거(split이라는 메서드사용)
let fruits="사과 , 딸기, 포도, 참외";
fruits = fruits.split(",");
console.log(fruits); 1~4까지 쪼개서 출력
console.log(fruits[3]); 세번째 과일만 출력
